# Student Management System

## Overview

Student Management System is a console-based Python application developed using Object-Oriented Programming (OOP) concepts. The system allows users to manage student records efficiently by performing operations such as adding, viewing, searching, updating, and deleting student information.

## Features

* Add new student records
* View all students
* Search students by Roll Number
* Update student information
* Delete student records
* Calculate grades automatically
* Display student statistics
* Menu-driven interface
* Object-Oriented Programming implementation

## Technologies Used

* Python 3
* OOP Concepts
* Lists and Loops
* Functions and Classes

## Project Structure

Student Management System

* Student Class
* StudentManagementSystem Class
* CRUD Operations
* Statistics Module
